#                                             ToDoReminder

TodoReminder is a small TodoApp based on modern Android application tech-stacks and MVVM architecture.



## Tech stack & Open-source libraries

- Kotlin based , Coroutines + Flow for asynchronous
- Hilt
- JetPack
  - Lifecycle
  - ViewModel
  - Room

- Architecture
  - MVVM

- CustomView
